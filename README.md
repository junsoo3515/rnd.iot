# IOT기반 스마트환경 모니터링 서비스
IOT기반 스마트환경 모니터링 서비스는 환경센서에서 수집한 데이터를 지도상에 표출하고 VMS와 연계해 CCTV 모니터링이 가능한 서비스입니다.

## 프로젝트 스택

### 클라이언트

이 프로젝트에서 사용되는 핵심 프론트 엔드 기술은 다음과 같습니다.

- HTML5
- CSS
- JS
- jQuery
- PHP
- GIS(지도 표출) 서비스
- VMS(CCTV 영상 표출 S/W)

### 서버

이 프로젝트에서 사용되는 핵심 백엔드 기술은 다음과 같습니다.

- Apache
- MySql
